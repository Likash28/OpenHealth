# Contributing to OpenHealth

Thanks for your interest in contributing!

## How to contribute

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes and test them
4. Commit: `git commit -m "feat: describe your change"`
5. Push and open a Pull Request

## What to work on

- New disease detection modules
- Model accuracy improvements
- UI/UX improvements
- Documentation

## Guidelines

- Keep ML model files out of the repo (use DVC or link externally)
- Add a requirements entry for any new dependency
- Include a brief description of your model's performance in the PR
